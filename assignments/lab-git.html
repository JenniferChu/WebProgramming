Lab: Git Introduction

Lab Objective

  1. Practice using Git and GitHub

Instructions

Please read the following instructions first, carefully!

  1. Create a GitHub account if you do not have one already at https://github.com/.
  2. Read https://help.github.com/articles/set-up-git on setting up Git on your computer.
    • Important Note 1: I do not recommend using the native app. It isn't as flexible as using the command line and you might as well get into the habit of typing and using the command line terminal.
    • Important Note 2: If you are using a Mac, I strongly recommend that you follow the Generating SSH Keys instructions. In the past, students have had lots of trouble using git credential-osxkeychain.
  3. Read https://help.github.com/articles/create-a-repo on creating a Git repository on GitHub, adding files to your repository, committing, and pushing files to GitHub. Please create the hello-world repository under your account!
  4. Make a second commit change. That is: find a funny image (e.g., using Google), add it to your hello-world repo, commit and push change to GitHub. Please also write short, one line, coherent commit messages (i.e., no "dr. steve brule" or "asdf" garbage).
    • Important Note 3: Please do not use photos of other students, me, or the TAs. That can be just downright creepy.

Assessment

This lab is worth 2 points.

  • (0.5 point) Successfully created the hello-world repo
  • (1 point) Successfully added, commited, and pushed a funny image to hello-world repo
  • (0.5 point) Short, one line, and coherent commit messages
