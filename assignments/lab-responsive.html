Web Programming

Lab: Windows (Responsive Design)

Lab Objectives

  1. Practice writing HTML.
  2. Practice writing CSS.
  3. Understand what responsive design is.
  4. Understand the engineering technique of using sprite sheets.

Instructions

Given the above image named homer.jpg and using only one HTML file named index.html and only one CSS file named responsive.css, create a web page where if you resize the browser screen width, a different part of the image will be rendered. That is:

  • If the browser screen width is greater than 1280 pixels, display the entire image homer.jpg (i.e., as seen above, with all 4 smaller pictures).
  • If the browser screen width is greater than 1024 pixels but less than or equal to 1280 pixels, display the first small picture of Homer Simpson (i.e., the one in the upper-left).
  • If the browser screen width is greater than 768 pixels but less than or equal to 1024 pixels, display the second small picture of Homer Simpson (i.e., the one in the upper-right).
  • If the browser screen width is greater than 480 pixels but less than or equal to 768 pixels, display the third small picture of Homer Simpson (i.e., the one in the bottom-left).
  • If the browser screen width is less than or equal to 480 pixels, display the last small picture of Homer Simpson (i.e., the one in the bottom-right).

The entire image homer.jpg is 600px x 600 px. You can assume that each of the 4 smaller pictures in it are 300px x 300px.

Getting Started

I am going to give you the HTML so you do not have to write it! Copy and paste the following HTML below into your index.html:

<!DOCTYPE HTML>

<html>

<head>
<title>Homer</title>
<link rel="stylesheet" href="responsive.css"/>
<meta charset="utf-8" />
</head>

<body>
	<div class="homer"></div>
</body>
</html>

Caveat: you are NOT allowed to modify the HTML!
