Web Programming

Lab: Where's the Train (MBTA Red Line)?

Lab Objectives

  1. Practice writing JavaScript.
  2. Practice using a third-party API.
  3. Practice parsing JSON data.
  4. Understand the limitations of client-side JavaScript.

Instructions

The MBTA provides a real-time schedule JSON API for the Red Line, Blue Line, and Orange Line subway trains. For the Red Line, the API is available at http://developer.mbta.com/lib/rthr/red.json. Notice in the API, the locations of a few trains are provided (the object that has a latitude "Lat" and longitude "Long"). In this lab, you are to create a web page that:

  1. Contains a Google Map. You will be using the Google Maps JavaScript API.
  2. Requests data from the MBTA Red Line real-time API using the XMLHttpRequest object, parses the JSON data, and displays on the map where the Red Line trains currently are. You will use client-side JavaScript to do all this.

Getting Started

Follow the Google Maps JavaScript API tutorial "Getting Started" at https://developers.google.com/maps/documentation/javascript/tutorial.

IMPORTANT NOTE: You do not need to create an API key for Google Maps. I also used the HTTP version of the API. See http://tuftsdev.github.com/WebProgramming/examples/google_maps/geolocation_map.html for example.
